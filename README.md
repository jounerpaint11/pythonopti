Самый простой способ
Общий гайд для запуска скриптов на линуксе:
Если файл называется script.py:
python script.py
или (чаще используется):
python3 script.py
Например:
python3 optimize.py
2️ Сделать файл исполняемым
Если в начале файла есть строка:
#!/usr/bin/env python3
тогда можно запускать как обычную программу.
Дать права на запуск:
chmod +x script.py
Запустить:
./script.py
3️ Если скрипт требует root (часто для оптимизации)
Некоторые скрипты меняют системные параметры, поэтому нужен sudo:
sudo python3 script.py
или
sudo ./script.py
4️Если Python не установлен
В Arch Linux обычно Python уже есть, но если нет:
sudo pacman -S python
 Важно:
Скрипты "для оптимизации системы", скачанные из интернета, лучше сначала открыть и посмотреть код:
nano script.py
или
less script.py
потому что они могут выполнять опасные команды.

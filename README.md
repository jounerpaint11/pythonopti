Общий гайд для запуска скриптов на линуксе: 
Файл называется opt1.py:
2️ Сделать файл исполняемым
Если в начале файла есть строка:
#!/usr/bin/env python3
тогда можно запускать как обычную программу.
Дать права на запуск:
chmod +x opt1.py
Запустить:
./opt1.py
3️ Если скрипт требует root (часто для оптимизации)
Некоторые скрипты меняют системные параметры, поэтому нужен sudo:
sudo python3 opt1.py
или
sudo ./opt1.py
4️Если Python не установлен
В  Linux обычно Python уже есть, но если нет:
sudo pacman -S python

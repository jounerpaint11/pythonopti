Краткое описание работы программы
Этот Python-скрипт предназначен для базовой оптимизации и проверки состояния Linux-системы.
При запуске программа выполняет несколько действий:
Проверяет состояние системы — выводит информацию о загрузке процессора, памяти и использовании диска.
Очищает временные файлы в директориях /tmp и ~/.cache, освобождая место на диске.
Ищет большие файлы (больше 100 MB) в каталоге /home, чтобы пользователь мог определить, какие файлы занимают много места.
Очищает системный кеш Linux (если программа запущена с правами root), что может освободить часть оперативной памяти.
Записывает отчёт о выполненных действиях в файл /tmp/system_optimization_report.txt.
Таким образом, программа помогает освободить место на диске, проверить состояние системы и выявить крупные файлы.
Общий гайд для запуска скриптов на линуксе: 
Файл называется opt1.py:
2️ Сделать файл исполняемым
Если в начале файла есть строка:
#!/usr/bin/env python3
тогда можно запускать как обычную программу.
Дать права на запуск:
chmod +x opt1.py
Запустить:
./opt1.py
3️ Если скрипт требует root (часто для оптимизации)
Некоторые скрипты меняют системные параметры, поэтому нужен sudo:
sudo python3 opt1.py
или
sudo ./opt1.py
4️Если Python не установлен
В  Linux обычно Python уже есть, но если нет:
sudo pacman -S python
